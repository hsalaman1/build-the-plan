# Build the Plan

An interactive behavior-strategy card game for ESE staff training. Teams read a student scenario, discuss it, then reveal the **Function** of the behavior and three matched strategies (**Proactive**, **Response**, **Replacement**) and score their answers.

## Play it
- **Live:** https://hsalaman1.github.io/build-the-plan/ (after GitHub Pages finishes building)
- Or download and open `index.html` in any browser.

## How it works
Set up teams, pick a discussion timer, choose which scenarios to play, then run each round: discuss, reveal the answer cards, and score. Includes a wild-card challenge, a print-friendly scoreboard, and a printable answer key. Runs entirely in the browser. No install, and no data is collected.
